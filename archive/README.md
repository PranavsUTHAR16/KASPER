# Archive Directory

This directory contains superseded draft implementations:
- `old_code/`: Early experimental prototypes and previous layer implementations.
- `new/`: Experimental draft modules (e.g. `kasper_composite_loss.py`, `kasper_kan1_regime_detection.py`, `kasper_kan2_regime_forecasting.py`).

The root-level files (`kasper.py`, `regime_detection.py`, `regime_forecasting.py`, `losses.py`, `train.py`, `evaluate.py`, `explain.py`, `preprocess_spy.py`, `spline_activation.py`) constitute the single canonical implementation of KASPER matching the TMLR (02/2026) paper.
